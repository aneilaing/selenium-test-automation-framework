# Selenium Test Automation Framework

This project demonstrates a UI test automation framework built using Java, Selenium WebDriver, and TestNG.

## Features
- Page Object Model (POM) design pattern
- Automated UI testing using Selenium
- Test execution with TestNG
- Positive and negative test scenarios

## Test Coverage
- Valid login test
- Invalid login test

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## How to Run
1. Clone the repository
2. Run `testng.xml` from IntelliJ or via Maven
